uPEG — clone target sniper

Read target's seed/traits from chain. Continuously poll the hook's current _randomSeed. On every poll, simulate _nextUpeg(Random(S, 0)) and check if the result would match the target. If a match appears, you have ~12s window to fire an LP removeLiquidity via Flashbots before any swap churns the seed.
Realistic odds: astronomically low for any specific target. This page exists to (a) prove the impossibility (b) catch you if lightning ever strikes.
