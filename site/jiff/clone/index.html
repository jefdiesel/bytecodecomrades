JIFF — clone target sniper

Two modes:
1. Clone — load a target JIFF id; on every poll, simulate the next mint and compare. If predicted traits match the target, fire an LP removeLiquidity via Flashbots before any swap rolls the seed.
2. Level rush — set a min level (e.g. L4). On every poll, simulate the next mint. If predicted level ≥ threshold, alert + fire your buy.
