JIFF — seed grinder

Exploits the deterministic seed in JIFFHook's _randomSeed (keccak of mostly-public inputs). Each pool→user transfer mints a JIFF with a level (0=common…4=rarest); this watches your wallet, decodes new mints, tells you keep or sell back.
v0 manual mode — you swap, this watches.

how to grind
  1. pick min level above (L3+ recommended)
  2. connect wallet → click start watching
  3. buy 1+ JIFF on the pool
  4. this page shows the resulting JIFF + level within ~6s
  5. if it matches → keep. if not → sell back (burns it, refunds most ETH)
